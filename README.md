# Dell Apex 15 — SDD Sandbox

A Spec-Driven Development test project: build a configurable laptop
product page (Dell-style PDP) entirely from the specs in `specs/`.

- **Start here** → [`specs/00-overview.md`](./specs/00-overview.md)
- **Data contract** → [`specs/01-data-model.md`](./specs/01-data-model.md)
- **Sample dataset** → [`data/laptops.sample.json`](./data/laptops.sample.json)
- **Bootstrap prompts** → [`BOOTSTRAP.md`](./BOOTSTRAP.md)

Hard constraint: all product data is loaded from a local JSON file
(`data/laptops.json`). No external APIs.
